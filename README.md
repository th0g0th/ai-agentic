# AI Agentic Skills

Reusable [Claude Code skills](https://docs.claude.com/en/docs/claude-code/skills.md) for agentic workflows.

## Skills

| Skill | Description |
|-------|-------------|
| [compound](compound/) | Extract learnings from completed work into searchable solution docs |
| [figma-reader](figma-reader/) | Read Figma files via MCP with token-efficient guided discovery |
| [prompt-leverage](prompt-leverage/) | Strengthen raw prompts into execution-ready instruction sets |

## Installation

Copy any skill folder to `~/.claude/skills/`:

```bash
cp -r <skill-name> ~/.claude/skills/
```
